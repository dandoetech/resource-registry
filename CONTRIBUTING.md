# Contributing

Thank you for considering contributing!

## Pull Requests
- Fork, create a feature branch, commit with clear messages.
- Add tests for new behavior.
- Ensure `composer qa` passes locally.

## Coding Style
- PSR-12 with strict types and trailing commas in multiline constructs.
- Run `composer cs:fix` before pushing.

## Security
- Please do not disclose security issues publicly. See `SECURITY.md`.
